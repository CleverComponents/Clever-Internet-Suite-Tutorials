# TCP client / server connection

The CustomSocket project is a code example for the [TCP client / server connection](https://www.clevercomponents.com/portal/kb/a68/tcp-client-server-connection.aspx) tutorial.   

The client establishes a connection and sends data. The server receives the data and echoes it back to the client.
