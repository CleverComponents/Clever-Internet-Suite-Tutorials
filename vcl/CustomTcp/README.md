# Custom TCP client / server

The CustomTcp project is a code example for the [Custom TCP client / server](https://www.clevercomponents.com/portal/kb/a69/custom-tcp-client-server.aspx) tutorial.   

A TCP client and a multithreaded TCP server that can send and receive multiple text lines as well as binary data.  


# Project folders

* ExchangeBinaryData — an example of sending and receiving Delphi TStream content;
* ExchangeTextLines — demonstrates how to send and receive multiple text lines using a Delphi TStrings object.
