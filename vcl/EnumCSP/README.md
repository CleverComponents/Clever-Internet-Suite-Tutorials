# Get CSP Information

The EnumCSP project represents a code example for the [Get CSP Information](https://www.clevercomponents.com/portal/kb/a58/get-csp-information.aspx) tutorial.   

Gets an information about installed Cryptographyc Service Providers.   


## Overview

This project shows how to programmatically enumerate Cryptographic Service Providers on a Windows system using native Windows CryptoAPI functions.

The project was compiled with Clever Internet Suite version 12.  
Tutorials and project files for previous versions of Clever Internet Suite can be found in the [Releases](https://github.com/CleverComponents/Clever-Internet-Suite-Tutorials/releases) section.


## Features

- Lists all installed CSPs
- Displays provider names and types
- Shows supported algorithms
- Useful for cryptographic diagnostics
